Se usan HashSet<string> para representar los conjuntos de ciudadanos.

Se generan 500 ciudadanos ficticios con nombres "Ciudadano 1" hasta "Ciudadano 500".

Se asignan 75 ciudadanos a Pfizer y 75 a AstraZeneca, con un rango que permite solapamiento (para simular quienes recibieron ambas dosis).

Se aplican operaciones de teoría de conjuntos:

ExceptWith → diferencia de conjuntos.

IntersectWith → intersección de conjuntos.

Se imprimen los listados solicitados.
